# Michael Sullivan Portfolio

A modern developer portfolio built with Next.js, React, and TypeScript showcasing frontend engineering projects, responsive UI design, and interactive user experiences.

## Live Demo

🔗 https://michaelsullivan-portfolio-next-2.vercel.app/

---

## Features

- Responsive modern UI
- Dynamic project pages
- Animated transitions with Framer Motion
- Interactive contact modal
- Smooth scrolling and hover interactions
- Mobile-first responsive layouts
- Optimized deployment with Vercel
- Reusable component architecture

---

## Tech Stack

- Next.js
- React
- TypeScript
- Framer Motion
- CSS Modules / Global CSS
- Vercel

## Challenges & Solutions

One challenge during development was maintaining smooth animations and responsive layouts across multiple screen sizes while keeping the UI performant and scalable.

To solve this:

- reusable layout patterns were introduced
- animations were optimized with Framer Motion
- responsive breakpoints were refined for mobile devices
- component structure was organized for scalability and maintainability

---

## Performance Optimizations

- Optimized image handling
- Reusable component architecture
- Responsive-first design approach
- Reduced unnecessary rerenders
- Organized and scalable project structure
